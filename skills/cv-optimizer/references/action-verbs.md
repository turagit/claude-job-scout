# Action Verbs for CV Bullets

**Leadership:** Championed, Directed, Led, Orchestrated, Spearheaded, Transformed
**Achievement:** Delivered, Exceeded, Secured, Solved, Surpassed
**Technical:** Architected, Automated, Built, Deployed, Engineered, Optimized, Scaled
**Analysis:** Analyzed, Benchmarked, Diagnosed, Forecasted, Quantified, Validated
**Growth:** Boosted, Expanded, Generated, Launched, Doubled, Tripled
**Process:** Eliminated, Modernized, Redesigned, Simplified, Standardized
**Communication:** Collaborated, Facilitated, Negotiated, Partnered, Presented

Start bullets with past-tense verbs (present for current role). Don't repeat verbs within a role. Pair with quantifiable results.
