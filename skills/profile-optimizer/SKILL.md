# LinkedIn Profile Optimizer

Analyze and enhance LinkedIn profiles to maximize visibility to recruiters, pass ATS keyword filters, and present a compelling professional brand.

## Profile Analysis Framework

Evaluate the profile across these sections:

### 1. Headline (Critical — first thing recruiters see)
- Maximum 220 characters — use every character
- Formula: `[Current Role] | [Key Specialty] | [Value Proposition/Keywords]`
- Include 2-3 searchable keywords recruiters actually search for
- Avoid generic titles like "Looking for opportunities" or "Open to work"
- **Good:** "Senior Full-Stack Engineer | React & Node.js | Building Scalable SaaS Platforms"
- **Bad:** "Passionate professional seeking new challenges"

### 2. About / Summary Section (High Impact)
- First 3 lines visible before "see more" — make them count
- Hook: Start with a compelling statement about what you do and the impact you create
- Structure: Hook → Expertise areas → Key achievements → What you're looking for
- Length: 200-400 words (enough for keywords, short enough to read)
- Include a call to action: "Let's connect" or "Reach out to discuss..."
- Weave in keywords naturally — never list keywords at the bottom

### 3. Experience Section
- Mirror CV achievements but adapt for LinkedIn's conversational tone
- Each role: 3-5 bullet points using PAR method
- Include rich media where possible (project links, presentations)
- Ensure job titles match industry-standard terminology (for search)
- Current role should be the most detailed

### 4. Skills & Endorsements
- Prioritize top 3 pinned skills — these appear most prominently
- Include 30+ skills (LinkedIn allows up to 50)
- Mix: technical skills, tools, methodologies, soft skills
- Pin the 3 most relevant skills for target roles
- Skills section feeds directly into LinkedIn's recruiter search algorithm

### 5. Additional Sections
- **Featured:** Pin best work samples, articles, or key posts
- **Certifications:** Include all relevant certifications with issue dates
- **Education:** Include relevant coursework, honors, activities
- **Recommendations:** Aim for 3+ from managers or senior colleagues
- **Volunteer/Projects:** Include if relevant to career goals

## LinkedIn SEO Strategy

LinkedIn's search algorithm prioritizes:

1. **Exact keyword matches** in Headline, About, and Experience
2. **Profile completeness** — All-Star profiles rank higher
3. **Activity signals** — Recent activity boosts search ranking
4. **Connection proximity** — 1st and 2nd degree connections appear first
5. **Location match** — Set location to where you want to work

### Keyword Optimization Process

1. Identify target job titles and their variations
2. Extract the top 20 keywords from 5-10 relevant job descriptions
3. Map each keyword to a natural placement in the profile:
   - Headline: 2-3 primary keywords
   - About: 5-8 keywords woven into narrative
   - Experience: 3-5 per role in achievement context
   - Skills: All relevant keywords as endorsable skills
4. Verify keywords appear at least 2-3 times across the profile (different sections)

## Profile vs CV Alignment

When both the CV and LinkedIn profile are available:

1. **Consistency check** — Dates, titles, and companies must match exactly
2. **Tone adaptation** — CV is formal and concise; LinkedIn is conversational and personal
3. **Content expansion** — LinkedIn can include more context, story, and personality
4. **Keyword sync** — Ensure all CV keywords also appear on LinkedIn
5. **Gap identification** — Flag any experience on CV missing from LinkedIn or vice versa

## Optimization Process

1. **Screenshot/read the current profile** via browser
2. **Score each section** (1-10) against the framework above
3. **Identify the 5 highest-impact changes**
4. **Write optimized content** for each section
5. **Present before/after comparison** for user approval
6. **Apply changes** via browser with user permission
