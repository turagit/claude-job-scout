# LinkedIn Profile Optimizer

Analyze and enhance LinkedIn profiles to maximize recruiter visibility and ATS compatibility.

## Profile Analysis Framework

### 1. Headline (Critical)
- 220 characters max — use all of them
- Include 2-3 searchable keywords recruiters actually use
- See `references/headline-formulas.md` for patterns

### 2. About / Summary
- First 3 lines visible before "see more" — hook must be strong
- Structure: Hook -> Expertise -> Key achievements -> What you're looking for
- 200-400 words, keywords woven naturally (never listed at bottom)
- End with call to action

### 3. Experience
- 3-5 bullets per role using PAR method, current role most detailed
- Job titles must match industry-standard terms (for search)
- Include rich media where possible (links, presentations)

### 4. Skills & Endorsements
- 30+ skills (up to 50), pin top 3 most relevant to target roles
- Mix: technical, tools, methodologies, soft skills
- Skills section feeds directly into recruiter search algorithm

### 5. Additional Sections
- Featured: pin best work samples/articles
- Certifications, Education, Recommendations (aim for 3+), Volunteer/Projects

## Profile vs CV Alignment

Dates, titles, and companies must match. Tone shifts: CV is formal/concise, LinkedIn is conversational. Ensure all CV keywords appear on LinkedIn. Flag any gaps between the two.

## Optimization Process

1. Read current profile via browser, score each section (1-10)
2. Identify 5 highest-impact changes
3. Write optimized content with before/after comparison
4. Apply changes via browser with user permission
