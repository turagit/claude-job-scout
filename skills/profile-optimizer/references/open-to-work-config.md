# Open to Work Configuration Guide

## Why This Matters

The "Open to Work" recruiter signal is the single highest-impact discoverability setting on LinkedIn. Many recruiters filter their search results to show only candidates who have this enabled. Without it, the profile is invisible to a significant portion of recruiter searches.

There are two modes:
- **Private (recruiter-only):** Only visible to LinkedIn Recruiter users. Recommended for most job seekers, especially those currently employed.
- **Public (green badge):** Visible to everyone. Use only if openly job searching and comfortable with current employer knowing.

## Setup Steps

1. Go to LinkedIn profile → click "Open to" button (below headline)
2. Select "Finding a new job"
3. Configure each field:

### Job Titles (Critical)
- Add 3-5 industry-standard titles that recruiters actually search for
- Derive from CV role titles, mapped to searchable equivalents (see `recruiter-search-patterns.md`)
- Example: CV title "Tech Lead" → add "Senior Software Engineer", "Engineering Manager", "Tech Lead", "Staff Engineer"
- Do NOT use creative titles — recruiters search for standard ones

### Location
- Set to the broadest area you'd accept work in
- For remote roles: add "Remote" AND your actual location (some recruiters still filter by location even for remote)
- For hybrid: add the specific city/region
- Multiple locations allowed — add all that apply

### Work Types
- Select all that apply: Full-time, Part-time, Contract, Freelance, Internship
- For freelancers: ensure "Contract" and "Freelance" are both selected
- This maps directly to recruiter filters

### Start Date
- "Immediately" / "Actively looking" — highest urgency signal for recruiters
- Choose based on actual availability
- "Open to opportunities" is the lowest urgency — use if passively looking

### Visibility
- **Recruiters only** — recommended for most users, especially if currently employed
- **All LinkedIn members** — adds the green #OpenToWork badge on profile photo
- Note: "Recruiters only" is not perfectly private — your current company's recruiters are excluded, but the signal is not guaranteed invisible to all colleagues

## Optimisation Tips

- Review and update job titles quarterly or when changing search targets
- Keep location current — stale location data means missed searches
- Enabling Open to Work can increase InMail from recruiters by 2-3x
- Pair with a strong headline and skills section for maximum impact — Open to Work gets you into the search results, but headline and skills determine whether the recruiter clicks
