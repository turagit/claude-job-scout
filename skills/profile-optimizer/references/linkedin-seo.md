# LinkedIn SEO & Algorithm Deep Dive

## How LinkedIn Search Works for Recruiters

LinkedIn Recruiter search ranks profiles based on:

1. **Keyword relevance** — Exact and partial matches in headline, about, experience, skills
2. **Profile completeness** — All-Star status profiles rank significantly higher
3. **Network proximity** — 1st > 2nd > 3rd degree connections
4. **Activity recency** — Profiles with recent activity rank higher
5. **Engagement signals** — Profile views, connection acceptance rate, response rate

## All-Star Profile Checklist

LinkedIn assigns profile strength levels. All-Star requires:
- Profile photo (professional headshot)
- Headline (customized, not default job title)
- About/Summary section (200+ words)
- Current position with description
- At least 2 past positions
- Education
- 5+ skills
- 50+ connections
- Industry and location set

## Keyword Placement Priority

Placement locations ranked by search weight:
1. **Headline** — Highest weight, always visible in search results
2. **Current job title** — Second highest, must use searchable terms
3. **About section** — High weight, allows natural keyword density
4. **Experience descriptions** — Medium weight, context for keywords
5. **Skills section** — Directly feeds recruiter skill filters
6. **Past job titles** — Lower weight but still indexed

## Keyword Density Guidelines

- Headline: 2-3 primary keywords
- About: Primary keywords 2-3 times, secondary keywords 1-2 times
- Each Experience entry: 3-5 relevant keywords
- Skills: 30-50 keywords as individual skill entries
- Total unique keywords across profile: 40-60 for comprehensive coverage

## Recruiter Search Filters

Recruiters commonly filter by:
- Job title (current and past)
- Company (current and past)
- Location
- Skills
- Years of experience
- Education
- Industry
- Profile language

Ensure the profile is optimized for each of these filter dimensions.

## Common Mistakes That Hurt Visibility

1. Using creative job titles that no recruiter searches for
2. Leaving the About section empty
3. Having fewer than 5 skills listed
4. Not including location or setting it to a non-target city
5. Having a default LinkedIn URL instead of a custom one
6. No profile photo (drastically reduces profile views)
7. Inconsistent job titles between CV and LinkedIn
8. Not listing technologies/tools in the skills section
