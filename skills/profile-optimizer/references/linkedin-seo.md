# LinkedIn SEO & Algorithm

## Recruiter Search Ranking Factors
1. **Keyword relevance** — exact/partial matches in headline, about, experience, skills
2. **Profile completeness** — All-Star profiles rank significantly higher
3. **Network proximity** — 1st > 2nd > 3rd degree
4. **Activity recency** — recent activity boosts ranking
5. **Engagement signals** — profile views, response rate

## All-Star Requirements
Photo, custom headline, about (200+ words), current position with description, 2+ past positions, education, 5+ skills, 50+ connections, industry and location set.

## Keyword Placement (by weight)
1. **Headline** — highest weight, always visible in search
2. **Current job title** — use searchable terms
3. **About section** — natural keyword density
4. **Experience descriptions** — context for keywords
5. **Skills section** — feeds recruiter skill filters

## Keyword Density
- Headline: 2-3 primary keywords
- About: primary 2-3x, secondary 1-2x
- Each experience entry: 3-5 keywords
- Skills: 30-50 as individual entries
- Total unique: 40-60 across profile

## Common Mistakes
Creative job titles no one searches for, empty About section, <5 skills, default LinkedIn URL, no photo, inconsistent titles between CV and LinkedIn.
